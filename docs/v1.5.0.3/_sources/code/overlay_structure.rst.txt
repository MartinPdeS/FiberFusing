Overlay Structure Module
------------------------

.. autoclass:: FiberFusing.base_fused.BaseFused
    :members:
    :inherited-members:
    :member-order: bysource

    The `BaseFused` class is a foundational class for handling the fusion of optical fiber structures. It provides methods for setting fusion degrees, adding fibers, and visualizing fused and unfused structures within the coordinate system.
